## 开始

### 小程序版本的 Vue.js

正如标题 **“小程序版本的vue.js”**, 这句话来自官网文档，官方介绍是下面这样的：

> mpvue （github 地址请参见）是一个使用 Vue.js 开发小程序的前端框架。框架基于 Vue.js 核心，mpvue 修改了 Vue.js 的 runtime 和 compiler 实现，使其可以运行在小程序环境中，从而为小程序开发引入了整套 Vue.js 开发体验。
